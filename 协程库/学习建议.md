# 学习建议

## <font style="color:#333333;">技能需求：</font>
这里给小伙伴们说一下学习本项目所需要花费的时间：

基础不错的：一天花5-6个钟，10天内也就是60h即可。

基础还差一点点的：一天花5-6个钟，20天内也就是120h即可

**总行数在：2000**

**注：对于学习能力好或者多花时间肯花时间的人，可能进度会比我说的更快**

## <font style="color:#333333;">c++基础要求：</font>
+ <font style="color:#333333;">c++11：掌握基本的语法概念，最好学习到了虚函数的动态多态，智能指针，函数指针（void(*func)(void)），lambda匿名函数，模板。（当然遇到了我会在下面举例子帮助更好的理解）</font>
+ <font style="color:#333333;">stl：掌握基本的容器使用如vector,set,map， function的函数对象等；</font>
+ <font style="color:#333333;">多线程：线程池的设计与使用，多线程资源竞争时候的互斥问题，信号量的操作等，线程的创建和结束、销毁；</font>
+ <font style="color:#333333;">计算机网络：网络编程socket，IO多路复用，高性能网络模式:Reactor和Proactor。</font>
+ <font style="color:#333333;">设计模式：单例模式主要是在最后FdManger的部分.</font>
+ <font style="color:#333333;">c++的初学者可以看：</font>[C++基础课](https://kamacoder.com/course.php?course_id=1)
+ stl: [STL基础课](https://kamacoder.com/course.php?course_id=10)
+ 计算机网络和操作系统：[图解网络介绍 | 小林coding](https://xiaolincoding.com/network/)
